# Online Test Portal

A modern, responsive, and user-friendly Online Test Portal built using HTML, CSS, and JavaScript. The portal allows students to register, log in, attempt MCQ-based tests, view results instantly, and download certificates after successful completion.

## Features

- Student Registration
- Student Login
- Secure Authentication
- Dashboard
- Multiple Subject Tests
- Timer-Based Exams
- Automatic Answer Evaluation
- Instant Result Generation
- Certificate Generation
- Responsive Design
- Search and Filter Tests
- Result History
- Score Calculation
- Local Storage Support

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/online-test-portal.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No additional installation or dependencies are required.
